Automatty — Unreal Engine Material Plugin

A C++ Unreal Engine plugin with a Python automation layer that streamlines the full material creation and assignment workflow. Automatty presents a texture import dialog, regex-matches filenames against standard PBR slot conventions (BaseColor, Normal, Roughness, Metallic, ORM, and more), and automatically builds material instances from a configurable base material — reducing what would otherwise be a repetitive manual process down to a single operation.

Built and compiled for UE5 (5.6.1) using RunUAT/BuildPlugin.

Houdini Python Tools

A suite of shelf tools and Python scripts that automate texture and material workflows inside Houdini.

Texture Auto-Mapper — Selects nodes, launches a file picker, then regex-matches texture filenames to shader parameters with fuzzy name matching. Handles UDIM sequences automatically, converting .1001. notation to .<UDIM>. on the fly.

UDIM Tag Standardizer — Enforces consistent UDIM tagging conventions across assets and projects.

Material Parameter Editor — A PySide2 UI tool (blue #0d3c87 theme) for batch-editing material parameters across a LOPs/Solaris material library.

Blender DH Toolkit

A custom Blender addon built around a context-sensitive pie menu, keymapped and configurable via addon preferences. The toolkit consolidates common artist operations into fast, muscle-memory-friendly access.

Includes a Weight Paint menu, Display menu (wireframe toggle, viewport visibility, shader editor shortcuts), viewport background controls, and a custom shortcut key setter. A custom icon system (load_icons) handles all iconography. The Material Shader Builder uses regex-based texture matching with user-defined naming pattern preferences.

Maintained across Blender 4.4 → 4.5, navigating API breaking changes introduced in 4.5.
